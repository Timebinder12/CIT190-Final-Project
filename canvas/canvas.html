This game is inspired by the old mobile game called "Cube Runner". The objective of this game is to navigate the center icon around the randomly generated cubes.

There are plans to update this game to have varying levels (easy, medium, and hard). Additionally, there should be a feature added when you get to a certain point score, the game automatically speeds up or the density of the cubes increases.
